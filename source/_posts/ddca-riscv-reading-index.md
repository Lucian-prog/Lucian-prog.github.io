这里整理《数字设计和计算机体系结构：RISC-V 版》的读书笔记索引。

## 记录目标

这组笔记不追求逐段复述，而是关注：

- 核心概念如何映射到硬件结构。
- 公式、时序和控制信号之间的关系。
- 章节知识点如何连接到 RTL 实验。
- 哪些地方容易在仿真、综合或理解中出错。

## 计划索引

| 主题 | 重点 |
| --- | --- |
| 数字抽象与二进制表示 | bit、补码、编码、溢出 |
| 组合逻辑 | 布尔表达式、门级结构、MUX、译码器 |
| 时序逻辑 | 触发器、寄存器、同步系统、时序约束 |
| HDL 基础 | 组合逻辑、时序逻辑、模块层次 |

| 数据通路 | ALU、寄存器堆、MUX、控制信号 |

| 流水线 | IF/ID/EX/MEM/WB、冒险与转发 |
| 存储系统 | Cache、局部性、命中率、性能分析 |
